文件準備：詳細清單

1. 環境調查表

下載表單

請提供以下資訊，以利評估 POC 環境需求：

  • 確定 POC 期間要進行測試的人員，並設定 DBSAFER 帳號進行控管。
  • POC 期間要納管的資料庫 (DB) 進行設定，以及哪些帳號有權限訪問納管的 DB，並針對某欄位去限制 SQL 查詢或遮罩設定。
  • POC 期間要納管的作業系統 (OS) 或終端機 (Terminal)，以及哪些帳號有權限訪問納管的 Terminal，並針對要管控的指令進行設定。

2. 建議測試功能 Check List

下載清單

請確認 POC 期間預計測試的功能項目：

  • 帳號管理的測試內容。
  • 資料庫存取權限的測試內容。
  • 控制伺服器存取權限的測試內容。
  • 日誌管理、報表功能、審核系統以及側錄功能的運用。

2. 白名單設定

請客戶將智雲對外 IP：60.250.145.142 加入白名單，以便工程師後續遠端連線進行設定。

3. 環境提供

請提供以下三個環境：

  • Client 端：Windows 作業系統環境／Win10 以上版本

    (可用另開 VM 的方式或者提供跳板機)

  • DBSAFER Server VM規格：CPU:4core／ RAM:8G ／100G硬碟

    地端VM：智雲工程師到現場上傳OVF檔

    (僅需確認server資源足夠，不需先建立VM，OVF已含作業系統RockyLinux 8.10)

    雲端VM：開機器時請選擇作業系統RockyLinux 8.10

  • Target Server：提供要由DBSAFER納管的測試環境OS或者DB。

    (OS：Linux／Windows Server 2008（含）以上的版本；DB：SQL/ NoSQL)

4. 網路設定

請提供 DBSAFER server 的網路設定：IP、NETMASK、GATEWAY。
